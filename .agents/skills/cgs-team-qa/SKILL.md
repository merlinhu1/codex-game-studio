# Codex Game Studio Team QA

Use this skill for team qa work in Template Game.

## Objective

Coordinate QA coverage, bug triage, regression, platform checks, and release evidence.

## Inputs

- AGENTS.md
- .codex/studio.json
- task-relevant files named by the user or task record
- design/gdd.md
- production/timeline.md
- task-relevant team files

## Procedure

1. Clarify the requested coordinate qa coverage, bug triage, regression, platform checks, and release evidence. and identify the current project stage.
2. Collect evidence for QA Scope, Regression, Bug Queue.
3. Coordinate handoffs between disciplines and name each owner, artifact, and validation need.
4. Produce the requested artifact or review with clear file paths and verification evidence.

## Write Targets

- production/
- design/

## Output Contract

- Summary
- QA Scope
- Regression
- Bug Queue
- Evidence
- Risks
- Changed files or proposed files
- Verification evidence
- Next owner or decision

## Quality Gates

- QA Scope
- Regression
- Bug Queue
- Evidence
- Scope remains bounded to the current task and project stage.
- Report labels unverified assumptions separately from evidence.

## Handoff

Report changed files, verification evidence, remaining risks, and the next owner or decision. Do not imply hidden hooks or autonomous background work.
